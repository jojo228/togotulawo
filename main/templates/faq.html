Une fois dans gestion de publication vous verrez le button publier, dans cette option vous verrer toutes les informations requises pour faire une publication.

Le délais minimum pour recevoir vos transactions est de 10 jours, toutes fois pour certaines raisons ca peut aller aù dela de ce qui est prévu. Merci de nous contacter sur le contact@togotulawo.com ou sur le 92 55 55 55.
